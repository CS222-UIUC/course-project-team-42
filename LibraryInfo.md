# Library Information

### Backend

- Python pipenv packaging tool  
```
pip3 install pipenv
```

- Python django framework
```
pipenv install django
```

- Python spacy
```
pip install -U pip setuptools wheel
pip install -U spacy
python -m spacy download en_core_web_sm
```

- Python numpy
```
pip install numpy
import numpy as np
```

### Test

- Python unittest 
```
pip install unittest

- Python wiki
```
pip install wikipedia
